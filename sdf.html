Space Defense Force


Space Defense Force (SDF) is a cool space combat game created with C# and MonoGame Framework, offering an exciting shoot 'em up experience. This project originated as an exercise for a school assignment, demonstrating the application of programming skills in game development.

Genre: Space combat, Shoot 'em up
Platform: PC

In this code I created the alien moving and shooting, every frame it makes a new random number (1 to 100) then checks if that number is 2 and the delay is 0, if all these conditions are met the alien will shoot.

For the movement it checks if the elapsed time is smaller then the delay and _right is true, if these conditions are met it will move the alien to the right by 14 in this case times the elapsed gametime in seconds, it will do the oposite when going to the left.
